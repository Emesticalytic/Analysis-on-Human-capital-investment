# Human Capital Investment: Post-16 Value-for-Money Analysis

**Learning Data Science | R Programming | Economics | Public Policy**

---

## Overview

This project investigates whether government spending on post-16 
education delivers positive value for money. Using publicly available 
data and HM Treasury Green Book methodology, I built a cost-benefit 
model in R comparing two Level 3 qualification pathways:

- A-levels (academic route)
- Level 3 Apprenticeships (vocational route)

---

## Key Question

> For every £1 developed country governments spend on post-16 
> education, how much economic value does society get back?

---

## Key Findings

| Pathway                 | Govt Cost | BCR   | Verdict      |
|-------------------------|-----------|-------|--------------|
| A-level (3 A-levels)    | £9,800    | 20.27 | Strong VfM   |
| Level 3 Apprenticeship  | £12,750   | 15.62 | Strong VfM   |

Both pathways deliver strong returns. Results remain robust across 
all sensitivity tests.
